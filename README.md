# Solomia Web Ultra-Premium 🍷

Bienvenido a **Solomia**, la web elegante y profesional para un bar o restaurante, totalmente **responsive**, con **menú filtrable**, **animaciones modernas**, **scroll parallax**, y **formulario de reservas conectado a Google Sheets**.

---

## 🌟 Características principales

- Diseño elegante, moderno y llamativo con tipografía sofisticada.  
- Menú filtrable por categorías: Entradas, Principales, Postres.  
- Hero con **efecto parallax** y animaciones de entrada (fade-in).  
- Formulario de reservas que se puede conectar automáticamente a **Google Sheets**.  
- Botón “Back to Top” elegante.  
- Responsive 100% (móvil, tablet, desktop).  
- Código **HTML/CSS/JS puro**, fácil de personalizar y subir a **GitHub Pages**.  
- Totalmente gratis y sin marca de agua.
